# AI-Enabled Autism Care Platform

An integrated AI-enabled platform for early autism screening,
diagnostic decision support, therapy planning, and longitudinal progress tracking.

## Tech Stack
- Backend: Django + Django REST Framework
- Frontend: Bootstrap / React
- AI/ML: scikit-learn
- Database: PostgreSQL
- Deployment: Render

## Core Modules
- Early Screening
- Diagnostic Support
- Therapy Planning
- Progress Tracking
